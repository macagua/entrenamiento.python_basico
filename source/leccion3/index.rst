Estructuras de datos en Python
==============================

En Python tiene como tipos de datos simples números: enteros,
de coma flotante y complejos, como pueden ser 3, 15.57 o 7 + 5j;
cadenas de texto, como "Hola Mundo" y valores booleanos: ``True`` 
y ``False``.
